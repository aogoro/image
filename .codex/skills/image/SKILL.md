# Генерация изображений (для Codex)

Тонкая обёртка для Codex. Весь канон — в `.agents/skills/image/REFERENCE.md`.

## Что делать

### 0. Определи `$SKILL_ROOT`

Найди расположение этого SKILL.md → поднимись на 3 уровня (`../../..`) → это `$SKILL_ROOT`.

### 1. Определи описание

Из `$ARGUMENTS` извлеки что пользователь хочет нарисовать. Если пусто — спроси.

### 2. Enhance prompt

Разверни краткое описание в детальный image prompt на английском (стиль, композиция, освещение, детали). 1-3 предложения.

### 3. Определи size

- `1024x1024` (default) — квадрат
- `1024x1536` — портрет (вертикальный)
- `1536x1024` — пейзаж (горизонтальный)

### 4. Сгенерируй

Используй image generation tool напрямую с enhanced prompt из шага 2.
Сохрани результат в `$SKILL_ROOT/.temp/image/<slug>.png`.
Сообщи пользователю абсолютный путь и размер файла.
